# FinSight — AI Financial Markets Agent

An intelligent AI agent that consolidates Indian stock market research, news, and mutual fund analysis into a conversational interface.

## Features

- 📊 Real-time stock analysis (prices, PE ratios, competitors)
- 📰 Market news with sentiment analysis
- 💰 Mutual fund NAV and holdings analysis
- 📈 Historical price trends and growth percentages
- 📄 Document analysis (PDFs, Excel, images)
- 💬 Conversational interface with memory
- ⭐ User feedback tracking & analytics

## Tech Stack

- **Backend:** FastAPI, Groq LLM (Llama 3.3 70B)
- **Frontend:** HTML/CSS/JavaScript
- **Data Sources:** yfinance, NewsAPI, MFAPI, World Bank API
- **Deployment:** Render
